# Шрифты

Положите сюда файлы **woff2** (можно woff):

| Файл | Шрифт |
|------|--------|
| `KudryashevDisplay-Regular.woff2` | Kudryashev Display |
| `SloopScript-Regular.woff2` | Sloop Script |

Допустимы альтернативные имена из `fonts.css`.

**Montserrat** подключается с Google Fonts в `index.html`.

Если файлов нет, заголовки и подписи временно отобразятся системным serif / cursive.
